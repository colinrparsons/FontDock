How It Works

  • Client Creation: Folders become clients automatically. "Client1" folder = "Client1" client.
  • Duplicate Detection: Fonts are checked by file hash (SHA256). Duplicates are updated, not re-added.
  • Metadata Extraction: Font names, styles, and family info extracted automatically.
  • Recursive: All subfolders are scanned for font files (.ttf, .otf, .ttc, .woff, .woff2).
  • Client Assignment: Fonts are tagged with their client for filtering and collections.
